##### 📖 회전 관련
#include <cstring> memmove(A, B, 크기) <br/>
→ 배열의 크기만큼 B를 A로 복사 (배열 복사할 때 유용한 함수) <br/>
<br/>
**2차원 배열 회전 공식** <br/>
좌상단 (a, b) <br/>
우하단 (c, d) <br/>
c-a == d-b를 만족해야 함 <br/>
- **시계방향 90도**: rotated[ **i** ][ **j** ] = origin[ **a + (d - j)** ][ **b + (i - a)** ] <br/>
- **반시계방향 90도**: rotated[ **i** ][ **j** ] = origin[ **a + (j - b)** ][ **d - (i - a)** ] <br/>
